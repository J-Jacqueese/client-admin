## 🎬 使用演示

### 查看模型详情

```
步骤：
1. 进入"模型管理"页面
2. 找到要查看的模型
3. 点击"查看"按钮
4. 从右侧滑出详情面板
5. 查看完整信息
6. 点击下载链接可直接访问
7. 点击关闭或点击遮罩层关闭
```

### 快速浏览多个项目

```
1. 点击第一个模型的"查看"
2. 查看完毕后关闭抽屉
3. 点击第二个模型的"查看"
4. 重复操作，无需跳转页面
```

## 🔧 技术实现

### Drawer 抽屉组件

```jsx
<Drawer
  title="模型详情"
  placement="right"
  width={720}
  onClose={() => setDetailDrawerOpen(false)}
  open={detailDrawerOpen}
>
  {/* 详情内容 */}
</Drawer>
```

### Descriptions 描述列表

```jsx
<Descriptions title="基本信息" bordered column={2}>
  <Descriptions.Item label="ID">{model.id}</Descriptions.Item>
  <Descriptions.Item label="名称">{model.name}</Descriptions.Item>
</Descriptions>
```

### 状态管理

```javascript
const [detailDrawerOpen, setDetailDrawerOpen] = useState(false);
const [viewingModel, setViewingModel] = useState(null);

const handleView = (model) => {
  setViewingModel(model);
  setDetailDrawerOpen(true);
};
```
